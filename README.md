# USB_TO_UART-CP2102
Simple USB to UART converter using CP2102 and USB-C, designed in KiCad

## Features
- USB to UART using CP2102
- USB-C input
- 0603 SMD passives

## Folder Contents
- `.kicad_pro` – Project file
- `.kicad_pcb` – PCB layout
- `.kicad_sch` – Schematic
- `gerbers/` – Gerber files for fabrication
